# 二分查找 binary search

## 基础模板

```java
public function binary_search(int[] a,int target)
{
    if(a = null || a.length() = 0)
        return -1;
    int left = 0;
    int right = a.length();
    int mid;
    while(left<=right) 
    {
        mid = (left+right)/2;
        if(a[mid]<target)
            left = mid+1; //多加1的作用是避免 target超出a数组的范围 造成死循环
        else(a[mid]>target)
            right = mid-1;
        else 
            return mid;
    }
    
}
```

## 找出最左侧的

```java
public function find_left(int a[],int target)
{
    int left = 0;
    int right = a.length();
    while(left<right-1)
    {
        mid = (left+right)/2;
        if(a[mid]==target)
            right = mid;
       	else if(a[mid]<target)
            left = mid;
        else
            right = mid;
    }
    
    if(a[left] == target)
        return left;
    else if(a[right] == target)
        return right;
    else 
        return -1;
}
```
